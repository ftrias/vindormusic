What is this?

JKnobMan is a 'Knob' graphic image design tool.
The image strip has continuous frames stitched vertical, stitched horizontal or individual files. Though JKnobMan is mainly designed for VST-plugin GUI development, it may be useful if you need image-strips like this.

JKnobMan is a freeware.
There are no limitations to use including commercial use.
Donations are appreciated if you would like.

http://www.g200kg.com/en/software/knobman.html (English)
http://www.g200kg.com/jp/software/knobman.html (Japanese)

Runnable on Java environment (Win/Mac/Linux)
At first, confirm the Java Runtime Environment (JRE) is installed. JRE7 or later is recommended.
http://www.java.com/

Packages

  • JKnobManXXX-mac.zip : Application bundle for Mac OS X.
  • JKnobManXXX-exe.zip : Wrapped to Windows .exe file. Though this version also need a JRE, it can run like a windows native application.
  • JKnobManXXX-jar.zip : Executable jar file. For any platform that JRE is available.
